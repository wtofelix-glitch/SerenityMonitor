## 5个项目详情

### 1️⃣ HermesKill - 安全急停
- **GitHub**: https://github.com/theopitori/hermeskill
- **描述**: 实时监控 tool call + LLM turn，runaway 立即 kill 并生成死亡证明
- **定位**: "生产环境终于敢放飛了！"
- **来源推文**: https://x.com/GitTrend0x/status/2061800087756636178 (Jun 2)

### 2️⃣ hermes-proficiencies - 职业技能包
- **GitHub**: https://github.com/sene1337/hermes-proficiencies
- **描述**: methodical-Hermes 专业技能包，workspace hygiene + PR 流程 + spec-driven 全覆盖
- **定位**: Agent 终于"职业"起来了！

### 3️⃣ shadow-cto - 仓库记忆（Shadow CTO）
- **GitHub**: https://github.com/pulkitg/shadow-cto
- **描述**: 持久化 GitHub 仓库记忆容器，自然语言问代码历史和决策理由
- **定位**: "整个 repo 变成 Agent 的长期记忆"

### 4️⃣ claude-store - Mind Transposition 灵魂转移
- **GitHub**: https://github.com/13331112522/claude-store
- **描述**: Mind Transposition 跨平台灵魂转移技能，personality/skills/memory 无缝迁移
- **定位**: "灵魂不丢"终于实现了！

### 5️⃣ mnemosyne - 高级本地记忆
- **GitHub**: https://github.com/AxDSan/mnemosyne
- **描述**: 本地混合记忆系统，hybrid search + sleep consolidation，召回更准幻觉更少
- **定位**: 记忆从"存"进化成"真懂你"
